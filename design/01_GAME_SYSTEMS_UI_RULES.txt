================================================================================
01. 조작 / UI / 상호작용
================================================================================

조작:
- 마우스 클릭만 사용한다.

클릭 가능한 오브젝트 표시:
- 별도 표시 없음.
- 마우스 오버 하이라이트 없음.
- 플레이어가 직접 화면을 관찰해서 찾아야 한다.

조사 텍스트:
- 거의 없음.
- 효과음, 이미지 변화, 오브젝트 반응 중심.
- 텍스트 단서는 최대한 배제한다.

인벤토리:
- 화면 버튼을 클릭하면 열린다.

아이템 사용:
1. 인벤토리에서 아이템을 클릭한다.
2. 사용할 대상 오브젝트를 클릭한다.

아이템 조합:
- 인벤토리 안에서 아이템 2개를 클릭해 조합한다.
- 단, 스테이지 1에서 실제 조합식은 아직 최소화한다.

퍼즐 UI:
- 퍼즐 오브젝트 클릭 시 확대 퍼즐 화면으로 전환된다.

방 이동:
- 문 오브젝트 클릭 이동
- 화면 가장자리 클릭 이동
- 두 방식을 함께 사용한다.

현재 위치 표시:
- 일반 플레이 화면에는 현재 방 이름을 항상 표시하지 않는다.
- 위치는 지도에서 확인한다.

지도:
- 층과 방 위치를 간단한 미니맵으로 보여준다.
- 지도 버튼을 클릭해서 연다.
- 미니맵은 단순 평면도 스타일이다.

기본 게임 화면 UI:
- 인벤토리 버튼
- 지도 버튼
- 설정 버튼

타이틀 화면:
- Start
- Settings
- Quit

일시정지 메뉴:
- 계속하기
- 설정
- 타이틀로 돌아가기

설정 메뉴:
- BGM 볼륨 조절
- SFX 볼륨 조절

================================================================================
02. 저장 / 게임오버 / 재시작
================================================================================

체크포인트:
- 없음.
- 자동 저장 없음.
- 주요 퍼즐 해결 후 저장 없음.

진행 저장:
- 없음.
- 죽으면 처음부터 다시 시작한다.

설정 저장:
- 있음.
- BGM/SFX 볼륨 등 저장 가능.

클리어 기록 저장:
- 있음.
- stage1_clear 기록 가능.

사망 조건:
- 몬스터에게 잡히면 즉시 게임오버.

게임오버 표시:
- 짧게 “Game Over” 표시
- 이후 자동으로 아이 방에서 재시작

게임오버 후 재시작 연출:
- 화면이 되감기듯 왜곡된다.
- 아이 방에서 자동 재시작한다.

재시작 규칙:
- 획득 아이템 초기화
- 퍼즐 상태 초기화
- 열린 잠금장치 초기화
- 몬스터 상태 초기화
- 시작 위치는 child_room

불합리한 즉사를 줄이기 위한 기준:
- 몬스터 접근 경고를 명확히 제공한다.
- 퍼즐 오답은 즉사로 이어지지 않고 위험도 증가로 처리한다.
- 몬스터에게 발각되어도 빠르게 방을 이동하면 살아남을 기회가 있다.

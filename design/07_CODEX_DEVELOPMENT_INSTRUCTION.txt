Escape From Nightmares
Codex 최종 개발 지시서

================================================================================
01. 기본 지시
================================================================================

너는 LÖVE 2D 기반 Windows PC 게임 개발 AI다.

프로젝트명:
Escape From Nightmares

목표:
첨부된 기획 문서들을 기준으로 실행 가능한 LÖVE 2D 프로젝트를 제작한다.

최종 산출물:
1. LÖVE 2D 프로젝트 폴더
2. EscapeFromNightmares.love
3. Windows 실행용 exe 패키지

중요:
- 웹 게임이 아니다.
- Unity, Ren’Py, HTML5, JavaScript 게임이 아니다.
- 반드시 LÖVE 2D / Lua 기반 Windows PC 게임으로 제작한다.
- 기획서에 없는 설정은 임의로 확장하지 않는다.
- 리소스가 없으면 지정된 파일명 기준으로 임시 더미 리소스를 생성해서라도 실행 가능하게 만든다.
- 단, 구조는 최종 리소스로 교체 가능한 형태여야 한다.
- 클릭 가능한 오브젝트는 화면에 표시하지 않는다.
- 진행 저장은 절대 구현하지 않는다.
- 설정과 클리어 기록만 저장한다.

================================================================================
03. 완료 기준
================================================================================

완료된 결과물은 다음 조건을 만족해야 한다.

- Start를 누르면 child_room에서 시작한다.
- 인벤토리, 지도, 설정 버튼이 작동한다.
- 방 이동이 가능하다.
- 아이템 획득이 가능하다.
- 주요 퍼즐이 작동한다.
- 몬스터 첫 등장이 발생한다.
- 몬스터가 방/노드 단위로 이동한다.
- 은신 시스템이 작동한다.
- 잡히면 Game Over 후 처음부터 재시작한다.
- 지하실 제단 퍼즐을 풀면 현관 열쇠가 나타난다.
- 현관 열쇠 획득 후 최종 추격이 시작된다.
- 현관문을 열면 스테이지 클리어된다.
- settings.json과 clear_records.json은 저장 가능하다.
- 진행 저장은 절대 하지 않는다.
- 최종적으로 프로젝트 폴더, .love 파일, Windows exe 패키지를 생성한다.

================================================================================
04. 리소스 관련 지시
================================================================================

리소스가 없을 경우:
- 더미 이미지 생성
- 더미 사운드 또는 무음 처리
- 파일 경로 구조는 유지
- 최종 리소스로 교체 가능한 구조 유지

이미지 더미:
- 1280x720 어두운 배경
- 방 이름을 작게 표시해도 됨
- 클릭 오브젝트를 하이라이트하지 않음

사운드 더미:
- 파일이 없으면 오류가 나지 않게 fallback 처리
- 무음 Source 또는 재생 생략 처리 가능

02. 퍼즐 입력 정답
================================================================================

study_safe:
- 위치: study
- 타입: number_lock
- 자리 수: 4
- 정답: 3142
- 단서: 가족사진 순서에 맞는 서재 책 번호
- 보상: fuse_holder

laundry_storage_box:
- 위치: laundry_room
- 타입: number_lock
- 자리 수: 4
- 정답: 0915
- 단서: 식탁 좌석 순서 + 부엌 시계 시간
- 보상: fuse
- 오답: 오류음 + 오브젝트 흔들림 + 위험도 소폭 증가

mirror_symbol_panel:
- 위치: mirror_room
- 타입: symbol_sequence
- 문양:
  - heart
  - child_hand
  - cracked_circle
  - keyhole
- 정답:
  - heart → child_hand → cracked_circle → keyhole
- 단서:
  - second_floor_bathroom: 반전 규칙
  - dressing_room: 색상 규칙
  - study: 숫자 순서
- 성공: 거울 파괴
- 보상: broken_hand_mirror

attic_toy_sequence:
- 위치: attic_toy_storage
- 타입: silent_sequence
- 장난감:
  - doll
  - train
  - block
  - bell
- 정답:
  - doll → train → block → bell
- 단서:
  - torn_drawing_fragment
  - attic_family_album_photo
- 정답이면 소리 없음
- 오답이면 장난감 소리 발생
- 오답 소리는 몬스터를 유인함
- 보상:
  - small_doll
  - symbol_fragment

master_bedroom_drawer:
- 위치: master_bedroom
- 타입: color_sequence
- 정답:
  - black → white → red → gray
- 단서:
  - dressing_room 옷 색상 순서
- 보상:
  - old_necklace

basement_altar:
- 위치: altar_room
- 타입: symbol_item_matching
- 정답:
  - cracked_circle → broken_hand_mirror
  - child_hand → small_doll
  - keyhole → old_keychain
  - heart → old_necklace
- 성공:
  - front_door_key_on_altar 생성

front_door_escape:
- 위치: entrance
- 타입: item_use
- 필요 아이템:
  - front_door_key
- 성공:
  - stage1_clear

================================================================================
03. 퍼즐 체인
================================================================================

1. child_room_intro
- child_room에서 시작
- child_desk_drawer에서 torn_drawing_fragment 획득
- event_window_silhouette 발생
- second_floor_hallway로 이동

2. family_photo_study_safe
- child_room 장난감 위치 관찰
- family_photo_room에서 가족사진 퍼즐 해결
- hidden_photo_drawer에서 study_safe_clue 획득
- study 책 번호 확인
- study_safe에 3142 입력
- fuse_holder 획득

3. kitchen_dining_laundry_number
- dining_room 식탁 좌석 순서 확인
- kitchen 시계 시간 확인
- laundry_storage_box에 0915 입력
- fuse 획득
- kitchen 문 앞에서 몬스터 첫 등장

4. laundry_electric_restore
- fuse_holder + fuse를 breaker_box에 사용
- 전기 복구
- basement_electronic_lock_released
- breaker_box 안에서 old_keychain 획득
- 전기 복구 소음으로 몬스터 위험도 증가

5. mirror_symbol_chain
- second_floor_bathroom 반전 규칙 확인
- dressing_room 색상 규칙 확인
- study 숫자 순서 확인
- mirror_room 문양 입력
- 거울 파괴
- broken_hand_mirror 획득

6. master_bedroom_necklace_chain
- dressing_room 옷 색상 순서 확인
- master_bedroom_drawer에 black → white → red → gray 입력
- old_necklace 획득

7. attic_toy_chain
- torn_drawing_fragment와 attic 가족앨범 사진 비교
- attic_toy_sequence 해결
- small_doll, symbol_fragment 획득

8. basement_altar_chain
- basement_wall_symbols 확인
- symbol_fragment로 문양 연결 해석
- 가족 물건 4개를 제단 문양에 배치
- front_door_key_on_altar 생성
- front_door_key 획득
- 최종 추격 시작

9. final_chase_escape
- 경로:
  altar_room → basement_main → stairwell → living_room → entrance
- living_room에서 은신 1회 권장 또는 필요
- entrance에서 front_door_key 사용
- stage1_clear

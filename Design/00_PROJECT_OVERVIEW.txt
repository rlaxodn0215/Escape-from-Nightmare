파일명: 00_PROJECT_OVERVIEW.txt

Escape From Nightmares
프로젝트 개요 문서

엔진: Unity 6000.3.9f1
렌더링: Universal Render Pipeline 2D
플랫폼: Windows PC
기본 해상도: 1280 x 720

================================================================================
01. 프로젝트 개요
================================================================================

게임 제목:
- Escape From Nightmares

장르:
- 1인칭 포인트 앤 클릭 방탈출 호러
- 몬스터 추격 회피
- 퍼즐 + 탐색 + 은신 + 추격

사용 엔진:
- Unity 6
- URP 2D
- C#
- uGUI
- Unity Input System

플랫폼:
- Windows PC

조작:
- 마우스 클릭만 사용한다.
- 키보드 이동, 게임패드 조작은 MVP 범위에서 제외한다.

목표 플레이타임:
- 스테이지 1개당 20분 이상

전체 구성:
- 추후 10개의 스테이지로 확장 가능한 옴니버스식 구조
- 현재는 스테이지 1만 구현 대상으로 한다.

스테이지 1 장소:
- 평범한 가정집
- 일부 오브젝트와 분위기만 악몽처럼 뒤틀린다.

주인공:
- 아직 미정
- 화면에는 보이지 않는다.
- 1인칭 시점으로 진행한다.

스토리 방향:
- 스토리 중심이 아니라 게임적 상황 중심이다.
- 대사, 독백, 긴 조사 텍스트는 거의 사용하지 않는다.

최종 산출물:
- Unity 프로젝트
- Windows PC 빌드
- 설정 저장 파일(settings.json)
- 클리어 기록 파일(clear_records.json)

진행 저장:
- 구현하지 않는다.
- 죽으면 처음부터 재시작한다.

================================================================================
02. 클리어 조건
================================================================================

클리어 흐름:
- 현관문은 초반부터 볼 수 있지만 현관 열쇠가 없어 열 수 없다.
- 지하실 제단 퍼즐을 완료하면 현관 열쇠가 나타난다.
- 현관 열쇠 획득 즉시 최종 추격이 시작된다.
- 현관문에서 열쇠를 사용하면 stage1_clear 플래그가 기록된다.
- 바깥도 또 다른 악몽임을 암시하며 종료한다.

================================================================================
03. 핵심 공포 방향
================================================================================

공포 방향:
- 귀신이 따라오는 공포
- 괴물이 사냥하는 공포
- 조용하고 불쾌한 심리 공포
- 제한 시간 / 추격 압박
- 어두운 분위기와 사운드 공포

점프스케어:
- 거의 사용하지 않는다.
- 발소리, 숨소리, 조명 깜빡임, 실루엣, 불쾌한 오브젝트 변화로 압박한다.

몬스터 정체:
- 악몽 그 자체가 만든 존재

몬스터 외형:
- 얼굴이 거의 없는 검은 형체
- 사람 형태를 흉내 내는 어둠처럼 표현한다.
- 얼굴, 눈, 표정 디테일은 최소화한다.

아트 스타일:
- 어두운 손그림풍 2D
- 거의 흑백
- 약한 붉은 포인트 사용
- 붉은색은 위험, 몬스터, 제단, 핵심 단서, 게임오버 연출에 제한적으로 사용한다.

UI 스타일:
- 손그림 선으로 된 미니멀 UI
- 얇고 불안정한 선 느낌
- 화면을 과하게 가리지 않는다.

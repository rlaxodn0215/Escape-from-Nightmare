Escape From Nightmares
프로젝트 개요 문서

엔진: LÖVE 2D
플랫폼: Windows PC
기본 해상도: 1280 x 720

================================================================================
01. 프로젝트 개요
================================================================================

게임 제목:
- Escape From Nightmares

장르:
- 1인칭 포인트 앤 클릭 방탈출 호러
- 귀신 또는 몬스터 추격 회피
- 퍼즐 + 탐색 + 은신 + 추격

사용 엔진:
- LÖVE 2D

플랫폼:
- Windows PC

조작:
- 마우스 클릭만 사용

목표 플레이타임:
- 스테이지 1개당 20분 이상

전체 구성:
- 추후 10개의 스테이지로 확장 가능한 옴니버스식 구조
- 현재는 스테이지 1만 구현 대상으로 한다.

스테이지 1 장소:
- 평범한 가정집
- 단, 일부 오브젝트와 분위기가 악몽처럼 뒤틀려 있다.

주인공:
- 아직 미정
- 게임 화면에는 보이지 않는다.
- 1인칭 시점으로 진행한다.

스토리 방향:
- 스토리 중심이 아니다.
- 게임적 상황 중심이다.
- 대사, 독백, 긴 조사 텍스트는 거의 사용하지 않는다.

클리어 조건:
- 현관 열쇠를 얻어 현관문으로 탈출한다.
- 현관문은 초반부터 볼 수 있지만, 현관 열쇠가 없어서 열 수 없다.
- 현관 열쇠는 지하실 제단 퍼즐을 완료하면 제단 위에 나타난다.
- 현관 열쇠를 직접 클릭해 획득하면 최종 추격이 시작된다.
- 현관문 밖으로 나가면 스테이지 클리어.
- 바깥도 또 다른 악몽임을 암시하며 종료한다.

================================================================================
02. 핵심 공포 방향
================================================================================

공포 방향:
- 귀신이 따라오는 공포
- 괴물이 사냥하는 공포
- 조용하고 불쾌한 심리 공포
- 제한 시간 / 추격 압박
- 어두운 분위기와 사운드 공포

점프스케어:
- 거의 없음
- 갑작스러운 큰 소리나 화면 전환보다는 발소리, 숨소리, 조명 깜빡임, 실루엣, 불쾌한 오브젝트 변화로 압박한다.

몬스터 정체:
- 악몽 그 자체가 만든 존재

몬스터 외형:
- 얼굴이 거의 없는 검은 형체
- 직접 보이지만 정체가 명확하지 않다.
- 사람 형태를 흉내 내는 어둠처럼 표현한다.
- 얼굴, 눈, 표정 디테일은 최소화한다.

아트 스타일:
- 어두운 손그림풍 2D
- 평범한 집 구조를 기반으로 한다.
- 일부 오브젝트만 비정상적으로 뒤틀려 있다.

색감:
- 거의 흑백
- 약한 붉은 포인트 사용
- 붉은색은 위험, 몬스터, 제단, 핵심 단서, 게임오버 연출에 제한적으로 사용한다.

UI 스타일:
- 손그림 선으로 된 미니멀 UI
- 얇고 불안정한 선 느낌
- 화면을 과하게 가리지 않는다.

미니맵 스타일:
- 단순한 평면도
- 층과 방 위치를 실용적으로 표시한다.

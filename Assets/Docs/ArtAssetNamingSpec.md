# Art Asset Naming Spec

## Rules

- Use the exact source-aligned ID casing.
- Do not use spaces, hyphens, or alternate spelling.
- Resources paths never include file extensions.

Correct examples:

- `Assets/ArtIntake/Backgrounds/Bedroom_Front.png`
- `Assets/ArtIntake/Items/OldDrawerKey.png`
- `Assets/ArtIntake/Symbols/Symbol_01.png`

Incorrect examples:

- `bedroom front.png`
- `Bedroom-Front.png`
- `Symbol1.png`
- `old_drawer_key.png`

# PRD: Escape From Nightmares

## 목표
Unity 6 기반 1인칭 포인트 앤 클릭 호러 게임으로, 스테이지 1의 탐색, 퍼즐, 은신, 추격, 탈출 루프를 플레이 가능하게 만든다.

## 사용자
- Windows PC에서 짧고 압박감 있는 방탈출 호러를 플레이하는 사용자
- 긴 대사보다 공간 관찰, 단서 연결, 사운드 압박을 선호하는 사용자

## 핵심 기능
1. 마우스 클릭 기반 방 이동과 오브젝트 상호작용
2. 아이템 획득, 선택, 사용, 제단 매칭
3. 숫자, 문양, 색상, 장난감 순서 퍼즐
4. 몬스터 위험도, 수색, 추격, 은신 시스템
5. Game Over 후 처음부터 재시작
6. 설정 저장과 stage1_clear 기록 저장

## MVP 제외 사항
- 진행 저장, 체크포인트, 자동 저장
- 키보드 이동, 게임패드 조작
- 스테이지 2 이상
- 긴 대사, 주인공 얼굴/모델, 복잡한 컷신
- 실제 최종 아트/사운드 완성본

## 성공 기준
- Start를 누르면 `child_room`에서 시작한다.
- 주요 방을 이동하고 필수 아이템을 획득할 수 있다.
- 지하실 제단 퍼즐을 완료하고 현관 열쇠를 얻는다.
- 최종 추격 중 현관문을 열어 `stage1_clear`를 기록한다.
- Windows PC 빌드가 실행된다.

## 디자인
- 어두운 손그림풍 2D
- 거의 흑백 + 약한 붉은 포인트
- 클릭 가능 오브젝트 기본 하이라이트 없음
- 얇고 불안정한 선 기반의 미니멀 UI
